# Cybersecurity Steganography

- ### Install Requirements

```bash
git clone https://github.com/proavipatil/CS-Steganography.git && cd CS-Steganography
```

```bash
pip install -r requiremnts.txt
```

- ### Run the Code

```bash
python steganography.py
```
